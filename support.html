Quick Start

  1. Open a Google Sheet
  2. Go to Extensions → Add-ons → Get add-ons
  3. Search for "Wallet Balance Tracker"
  4. Click Install and grant permissions
  5. A Balances menu will appear in the menu bar
  6. Go to Balances → Setup / Create Dashboard — do this once
  7. Open any wallet sheet, add wallet addresses in column B (row 3+)
  8. Add a chain name in row 1 col C and its RPC URL in row 2 col C
  9. Click Balances → Fetch ALL chains - this sheet
⚠️ You only need public wallet addresses (starting with 0x). Never enter private keys or seed phrases.

Sheet layout

  • Row 1, col C+ — chain name (e.g. "Ethereum")
  • Row 2, col C+ — RPC URL (e.g. https://eth.llamarpc.com)
  • Row 3+, col B — wallet addresses (0x...)

To add another chain: new column, row 1 = name, row 2 = RPC URL. That's it.

Common Issues

❌ "Cannot connect to RPC endpoint"

  • Verify the RPC URL is correct and includes the full path (e.g. https://mainnet.infura.io/v3/YOUR_KEY)
  • Check that the RPC provider is online — visit their status page
  • Try a different provider: Infura, Alchemy, QuickNode, or a free public RPC
  • Make sure the RPC supports the chain you're querying (an Ethereum RPC won't work for Polygon)

❌ "Balance not updating"

  • Use Balances → Fetch ALL chains to trigger a manual fetch
  • Check the wallet address format — must start with 0x and be exactly 42 characters
  • Verify the wallet exists on the chain you're querying
  • Free RPC tiers have rate limits — wait 10–30 seconds and retry

❌ "Wrong balance shown"

  • Confirm you're querying the right network (mainnet vs testnet)
  • RPC nodes can lag — try a different provider
  • Always verify on a blockchain explorer: Etherscan, Polygonscan, etc.

❌ "Permission denied"

  • Revoke and re-grant permissions via Extensions → Add-ons → Manage add-ons
  • Sign out of Google and sign back in, then retry

❌ No "Balances" menu after install

  • Refresh the page — the menu appears after page reload post-install
  • Make sure the add-on is installed for your account, not just the sheet

FAQ

Is my data stored on your servers?

No. Everything stays in your Google Sheet. The add-on runs via Google Apps Script — no data passes through servers we control. See the Privacy Policy for full details.

Which blockchains are supported?

Any EVM-compatible chain with a public RPC endpoint — Ethereum, Polygon, BSC, Arbitrum, Optimism, Avalanche, Base, and more. You configure the RPC, so you control which chains you query.

Can I track multiple wallets?

Yes. Add as many public wallet addresses as you need in column B from row 3 onwards.

Can I track multiple chains?

Yes. Each column from C onwards is one chain. Add as many columns as you need.

What balances does this show?

Native coin balances only — ETH on Ethereum, MATIC on Polygon, BNB on BSC, and so on. It does not fetch ERC-20 token balances.

Do I need a paid RPC plan?

No. Free tiers from Infura, Alchemy, or public RPCs work fine for personal use. Paid plans give you higher rate limits if you're tracking many wallets.

Is this add-on free?

Yes, completely free. No subscription, no paywalls.
