💡 Not sure what some words mean? Check the quick glossary below before you start — it explains the only three things you need to know.
Wallet address (0x…)

Your public crypto address — safe to share, looks like 0xAbc…1234. This is the only thing you paste into the Sheet. Never paste your private key or seed phrase.

RPC URL

A free web address that lets the Sheet talk to the blockchain and read balances. Think of it as a phone number for the network. Free URLs are listed later on this page.

Apps Script

Google's built-in scripting tool hidden inside Google Sheets. You only need it for Option B — for Option A it's already set up inside the copied Sheet.

EVM chain

Any blockchain compatible with Ethereum (Ethereum, Polygon, BSC, Arbitrum, Optimism, Base…). If it has an RPC URL, this tracker works with it.

1
Make a copy of the Sheet

Click the button below. Google will ask you to sign in and then copy the Sheet into your own Google Drive. You are not sharing anything with us — it's just a copy.

📋 Make a Copy — Google Sheets™ →

2
Open the copy & wait for the menu

After Google opens the copied Sheet, wait 5–10 seconds. A menu called ⚡ Balances should appear in the top menu bar next to "Help".

If you don't see it after 10 seconds, do a full page refresh (Ctrl+R / Cmd+R).

⚠️ The first time you run any Balances menu item, Google will ask you to Authorize the script. Click Review permissions → your account → Allow. This is normal — Google just wants you to confirm you trust the script in your own account.
3
Run Setup / Create Dashboard

Click ⚡ Balances → Setup / Create Dashboard. This creates a PORTFOLIO summary sheet. You only need to do this once.

4
Add a chain sheet for each blockchain you use

Click ⚡ Balances → Add chain sheet → type a name like Ethereum or Polygon → press OK.

Repeat this for every chain you want to track. Each chain gets its own sheet tab.

5
Add your RPC URL in cell B2

Click on the new chain sheet tab. Click on cell B2 and paste the RPC URL for that chain. Copy one from the list below:

  • Ethereum → https://eth.llamarpc.com
  • Polygon   → https://polygon-rpc.com
  • BSC       → https://bsc-dataseed.binance.org
  • Arbitrum → https://arb1.arbitrum.io/rpc
  • Optimism → https://mainnet.optimism.io
  • Base     → https://mainnet.base.org

More chains at chainlist.org — search the network name and copy any RPC URL listed there.

💡 B2 is just one cell — it is in column B (second column), row 2. Click that cell, paste the URL, press Enter.
6
Set a coin name in row 1 (column C)

Click cell C1 and type the coin ticker for this chain — for example ETH, POL, or BNB. Leave C2 blank (blank = native coin balance).

That's all the column needs. You can add more token columns later using the menu.

7
Paste your wallet addresses in column B, starting at row 3

Each public wallet address goes in its own cell in column B. Start at B3 and go down. Addresses start with 0x and are 42 characters long.

⚠️ Only enter your public wallet address (starts with 0x). Never paste a private key or seed phrase into any cell.
🎉
Fetch your balances!

Click ⚡ Balances → Fetch this sheet. Within a few seconds, live balances will appear in column C beside each wallet address.

Run the same menu item any time you want updated numbers.

🏆

You're all set for Option A!

You can now track balances across multiple wallets and chains. See the "What's next?" section below.

Option B

Add to your existing Sheet

ℹ️ This path requires opening Apps Script (Google's built-in editor). It takes about 5 minutes. If that sounds daunting, go back and use Option A instead.
1
Open your Google Sheet

Open the Sheet where you want to add wallet tracking.

2
Open Apps Script

In the top menu: Extensions → Apps Script. A new browser tab will open showing a code editor.

3
Add the WalletTracker library

In the left sidebar, click the + next to Libraries.

Paste this Script ID into the box that appears:

1yM1l94RPOknc3QMz7l4eRnNRHRFR0vow0sIEmYBh22nZ_kUBbfreBee1

Click Look up. Set the Identifier field to exactly WalletTracker (capital W, capital T). Click Add.

⚠️ The identifier must be WalletTracker with that exact capitalisation. Typing it differently causes a ReferenceError and nothing will work.
4
Replace Code.gs with the wrapper functions

Click Code.gs in the left sidebar. Select all the existing code (Ctrl+A / Cmd+A) and delete it. Then paste the entire block below:

function onOpen() { WalletTracker.registerMenu(); } function onInstall() { WalletTracker.registerMenu(); } function wt_fetchCurrentSheet() { WalletTracker.fetchCurrentSheetAllChains(); } function wt_fetchAllSheets() { WalletTracker.fetchAllSheetsAllChains(); } function wt_fetchOneChain() { WalletTracker.promptFetchOneChain(); } function wt_startAutoFetch() { WalletTracker.startAutoFetch(); } function wt_stopAutoFetch() { WalletTracker.stopAutoFetch(); } function wt_triggerFetch() { WalletTracker.triggerFetch(); } function wt_setupDashboard() { WalletTracker.setupDashboard(); } function wt_addWalletSheet() { WalletTracker.addWalletSheet(); } function wt_addTokenColumn() { WalletTracker.addTokenColumn(); } function wt_clearDecimalsCache() { WalletTracker.clearDecimalsCache(); } function wt_setFallbackRpc() { WalletTracker.setFallbackRpc(); } function wt_refreshPortfolio() { WalletTracker.refreshDashboardSummary(); } function wt_clearAllCache() { WalletTracker.clearAllCache(); } function wt_help() { WalletTracker.showHelp(); }
5
Save and close Apps Script

Press Ctrl+S (Windows) or Cmd+S (Mac) to save. Then close the Apps Script tab and go back to your Sheet.

Do a full page refresh (Ctrl+R / Cmd+R). The ⚡ Balances menu should now appear in the menu bar.

6
Continue with Option A steps 3 – 7

Your Sheet is now wired up. Follow Option A steps 3 – 8 above to run Setup, add a chain sheet, paste your RPC URL and wallet addresses, and fetch your first balances.

What's next?

Once your first balances are showing, here are some things you can do:

🪙
Track ERC-20 tokens

Use ⚡ Balances → Add ERC-20 token to track USDC, USDT, WETH and more.

📊
Portfolio summary

Run ⚡ Balances → Refresh Portfolio for a SUMIF total across all chains.

🔁
Auto-fetch 500+ wallets

Use ⚡ Balances → Start auto-fetch to run in the background every minute.

📡
Add a fallback RPC

Set a backup RPC so balances keep fetching even if your main provider has issues.

💡 Hover any balance cell after fetching to see a tooltip showing how much the balance changed since the last run — great for spotting wallet activity at a glance.
